# QuickFin Dashboard

## 🚀 GitHub Pages 零基础拖拽部署操作步骤

### 1. 登录 GitHub
1. 打开 https://github.com
2. 点击右上角【Sign in】，输入账号密码登录

### 2. 创建新仓库
1. 登录后，点击右上角【+】→【New repository】
2. **Repository name** 填写：QuickFin-Dashboard-1259278462
3. 其他选项保持默认（Public），不用勾选 README
4. 点击【Create repository】

### 3. 上传项目文件（拖拽方式）
1. 进入刚创建的仓库页面
2. 点击【Add file】→【Upload files】
3. 打开本地 QuickFin-Dashboard-1259278462 文件夹，把所有文件（index.html、script.js、style.css、README.md）全部选中，拖拽到网页上传区域
4. 等待上传完成，填写【Commit changes】（如“init”），点击绿色按钮【Commit changes】

### 4. 启用 GitHub Pages
1. 仓库页面点击【Settings】
2. 左侧菜单点击【Pages】
3. 【Build and deployment】选择 `Deploy from a branch`
4. Branch 选 `main`（或 `master`），文件夹选 `/ (root)`
5. 等待几秒，页面下方会出现绿色提示，显示你的 GitHub Pages 地址，例如：
   https://你的用户名.github.io/QuickFin-Dashboard-1259278462

### 5. 访问你的网页
- 复制上面链接，在浏览器打开即可看到你的网页

### 6. 常见问题
- **打不开？** 等 1-2 分钟再刷新
- **样式错乱？** 检查所有文件都已上传且无多余文件夹嵌套

---
全程无需命令行，只需浏览器操作和拖拽！
